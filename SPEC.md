# 血压记录网站规格说明书

## 1. 项目概述

### 项目名称
血压记录管理系统 (Blood Pressure Tracker)

### 项目类型
Web单页应用 (SPA)

### 核心功能
一款专业、安全、易用的血压记录管理网站，支持血压数据的记录、统计、分析和可视化展示，帮助用户更好地监测和管理自身健康状况。

### 目标用户
- 需要定期监测血压的人群
- 高血压患者及其家属
- 注重健康管理的用户

## 2. UI/UX 规格

### 2.1 布局结构

#### 页面组成
- **顶部导航栏** (固定高度: 64px)
  - Logo + 应用名称 (左侧)
  - 导航菜单 (中间)
  - 用户操作按钮 (右侧: 导出/导入/设置)

- **侧边功能菜单** (可折叠, 宽度: 240px/60px)
  - 记录血压
  - 历史记录
  - 数据统计
  - 图表分析
  - 数据管理

- **主内容区域** (响应式宽度)
  - 桌面端: 侧边栏 + 主内容 (3:9)
  - 平板端: 侧边栏可折叠 + 主内容
  - 移动端: 底部导航 + 主内容

#### 响应式断点
- 桌面端: ≥1200px (完整布局)
- 平板端: 768px - 1199px (可折叠侧边栏)
- 移动端: ≤767px (底部导航栏)

### 2.2 视觉设计

#### 色彩方案 (医疗健康风格 - 蓝色系)
```
--primary: #1976D2        /* 主色 - 医疗蓝 */
--primary-light: #42A5F5  /* 浅蓝 */
--primary-dark: #0D47A1   /* 深蓝 */
--secondary: #FF9800      /* 橙色 - 晚间记录 */
--secondary-light: #FFB74D /* 浅橙 */
--success: #4CAF50        /* 成功 - 绿色 */
--warning: #FFC107        /* 警告 - 黄色 */
--danger: #F44336         /* 危险 - 红色 */
--background: #F5F7FA    /* 背景色 */
--surface: #FFFFFF        /* 卡片表面 */
--text-primary: #212121   /* 主文字 */
--text-secondary: #757575 /* 次要文字 */
--border: #E0E0E0         /* 边框色 */
```

#### 早晨/晚上记录颜色区分
- 早晨记录: #1976D2 (蓝色系)
- 晚上记录: #FF9800 (橙色系)

#### 收缩压/舒张压线型区分
- 收缩压 (Systolic): 实线 (-)
- 舒张压 (Diastolic): 虚线 (--)

#### 字体
- 主字体: "Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif
- 数字字体: "Roboto Mono", monospace
- 标题: 24px/20px/18px (h1/h2/h3)
- 正文: 16px
- 辅助文字: 14px/12px

#### 间距系统
- 基础单位: 8px
- 间距等级: 4px, 8px, 12px, 16px, 24px, 32px, 48px

#### 视觉效果
- 卡片阴影: 0 2px 8px rgba(0,0,0,0.08)
- 悬停阴影: 0 4px 16px rgba(0,0,0,0.12)
- 圆角: 8px (卡片), 4px (按钮/输入框)
- 过渡动画: 0.2s ease-in-out

## 3. 功能规格

### 3.1 数据记录功能

#### 记录表单字段
| 字段 | 类型 | 必填 | 范围/格式 | 说明 |
|------|------|------|-----------|------|
| 记录类型 | radio | 是 | morning/evening | 早晨/晚上 |
| 收缩压 | number | 是 | 90-180 mmHg | |
| 舒张压 | number | 是 | 60-120 mmHg | |
| 心率 | number | 是 | 40-160 bpm | |
| 备注 | textarea | 否 | 最多200字符 | 可选 |
| 时间戳 | datetime | 系统生成 | YYYY-MM-DD HH:mm | 精确到分钟 |

#### 输入验证规则
- 收缩压: 90 ≤ value ≤ 180
- 舒张压: 60 ≤ value ≤ 120
- 心率: 40 ≤ value ≤ 160
- 备注: 0 ≤ length ≤ 200
- 所有数值字段必填

#### 错误提示
- 输入框下方红色文字提示
- 提交时滚动到第一个错误位置
- 错误字段边框变红 (#F44336)

### 3.2 CRUD功能

#### Create (创建)
- 提交成功后显示绿色Toast: "记录已保存"
- 2秒后自动刷新列表
- 自动滚动到新记录位置

#### Read (读取)
- 分页: 每页显示10/20/50条记录
- 筛选条件: 日期范围、记录类型
- 排序: 按时间升序/降序

#### Update (更新)
- 点击记录进入编辑模式
- 保留原始时间戳
- 显示修改记录标识

#### Delete (删除)
- 显示确认对话框: "确定要删除这条记录吗？"
- 删除后显示Toast: "记录已删除"
- 支持批量删除

### 3.3 数据统计功能

#### 时间维度
- **近7天**: 按日聚合 (每日平均、最高、最低)
- **近1个月**: 按周聚合 (每周平均及趋势)
- **近1年**: 按月聚合 (每月平均及波动)

#### 统计指标
- 平均值 (Mean)
- 最高值 (Max)
- 最低值 (Min)
- 标准差 (Std)

### 3.4 可视化功能

#### 图表类型
- 折线图 (趋势展示)
- 柱状图 (对比展示)

#### 图表维度
- 早晨/晚上分开显示 (蓝色/橙色)
- 收缩压/舒张压对比显示 (实线/虚线)

#### 交互功能
- 鼠标悬停: 显示详细信息 (时间、数值、心率、备注)
- 点击数据点: 弹出详情卡片
- 时间范围缩放: 支持拖拽选择
- 导出功能: CSV、PNG格式

### 3.5 数据存储方案

#### 本地存储 (LocalStorage)
- 存储用户血压记录
- 存储用户偏好设置
- 自动保存机制

#### 数据导入/导出
- 导出格式: JSON、CSV
- 导入格式: JSON、CSV
- 数据校验机制

#### 备份提醒
- 30天未导出提醒
- 每次打开应用检查备份时间
